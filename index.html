HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Declarations contain two important components: the CSS property we want to apply and the value of the property
  • A margin property indicates how much space we want around the outside of an element
  • The padding property adds space around the content inside an element
  • a link element creates the connection to the external file
  • The rel attribute specifies the relationship between the current document and the linked document resource
  • The href attribute specifies the location (URL) of the external resource. The href attribute is using a relative path
  • The period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m: -m is used to associate a message with our commit

JavaScript

  • A variable is a named container that allows us to store data in our code
  • To declare a variable we can use the keyword var and assign with =
  • Strings are contained with quotation marks
  • Each line of JavaScript code ends with a semicolon
  • Console.log method can be used to output a message to the web console by adding an argument
  • Chrome DevTools can be accessed on a webpage by right clicking, selecting inspect and clicking the console tab
  • Control flow is the order in which a computer executes code in a script
  • JavaScript is typically executed from top to bottom
  • Conditional statements such as if statements allow us to interrupt control flow
  • An array is a single variable used to hold a group of data
  • Arrays are zero-indexed and sequential, the first item of an array starts with 0 then continues to 1,2,3 etc
  • Functions give basic instructions for the computer to perform a specific task
